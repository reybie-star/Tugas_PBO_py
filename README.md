## Fitur Utama
Program ini memiliki fitur dasar untuk mengelola koleksi buku secara interaktif:
1. **Lihat Buku**: Menampilkan seluruh daftar buku yang ada beserta statusnya.
2. **Tambah Buku**: Menambahkan judul dan penulis buku baru ke dalam sistem.
3. **Pinjam Buku**: Mengubah status buku dari "Tersedia" menjadi "Dipinjam".
4. **Kembalikan**: Mengubah status buku kembali menjadi "Tersedia".
5. **Keluar**: Menghentikan program.
